hGeoPattern

Collection of RenderMan plugins for sampling Houdini geometry.

PxrSurface interior extinction driven by distance to curves.

Displacement of inner surfaces and cutting edges based on the distance to the outside and inside geometry groups.

Distance to the spline can be used as source of masks as well as parameter to construct uv coordinates along curves and cages.

Normals blending based on proximity to another surface.

Point cloud based texture bombing. Scattered points attributes like normals, directions, pscale can be used to build uv coordinates and manipulate scale, orientation and blending.

Point cloud based texture bombing allows artistic control over parameters such as direction and scale.
