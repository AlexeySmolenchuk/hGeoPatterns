### [HOME](../Readme.md) / [Reference](Reference.md) / filterArrayData

OSL shader

Perform filtering on values in **[ArrayData](../osl/include/hGeoStructsOSL.h)** sampled from **[samplePoints](samplePoints.md)** and **[readAttribute](readAttribute.md)** plugins.
Weighted, by distance same way as **[buildCoords](buildCoords.md)** blending.

If you need filtering compatible with VEX use **[pointCloudFilter](pointCloudFilter.md)**.
